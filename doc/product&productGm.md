# productGm & product 

 * productGm = product General manage = 商品
 * product = 款式

結合方式為

productGm + product
1: n

一個商品裡面可以有多個款式，顯示方式為

```
productGM (product)
```

e.g.

```
男裝 牛津襯衫(黃)
男裝 牛津襯衫(紅)
```

### productGm:
裡面會列管關於商品的『總』資訊，商品描述，產地，對應的品牌，對應的館別

### product:
裡面會詳細列出，品項裡面需要的資訊，例如，
商品編號，價格，庫存 ...
